# Distributed Scalar-on-Function Regression

This repository contains the complete code used in the paper 
**"On Distributed Estimation for Scalar-on-Function Regression Models."**

The repository includes the following files:

- **MCSimulation.R**  
  Main script for conducting the Monte Carlo simulation studies.

- **real_data.R**  
  Main script for the empirical analysis using real data.

- **interval_score.R**  
  Function for computing the interval score used in evaluating prediction intervals.

- **generate_data.R**  
  Function for generating simulated data according to the data-generating processes described in the paper.
